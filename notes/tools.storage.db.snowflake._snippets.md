### Checking for Data Freshness and Volume ###
``` SQL
-- For Normal Tables
SELECT
    TABLE_CATALOG,
    TABLE_SCHEMA,
    TABLE_NAME,
    ROW_COUNT,
    BYTES,
    CONVERT_TIMEZONE('UTC', CREATED) as CREATED,
    CONVERT_TIMEZONE('UTC', LAST_ALTERED) AS LAST_ALTERED
FROM DEV_FDWANALYTICS.INFORMATION_SCHEMA.TABLES
WHERE TABLE_SCHEMA NOT IN ('INFORMATION_SCHEMA')
AND TABLE_TYPE NOT IN ('VIEW', 'EXTERNAL TABLE')
ORDER BY TABLE_CATALOG, TABLE_SCHEMA, TABLE_NAME

-- For External Tables
SHOW EXTERNAL TABLES
```
